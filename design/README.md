# Design

Папка для макетов/ассетов:

- Ссылки на Figma, экспортированные SVG/PNG/WebP.
- UX-флоу: регистрация/логин, карта, карточка места, конструктор маршрутов, отзывы, сообщения, лента, чат-бот.
- Токены дизайна: цветовая палитра, типографика, сетка, состояния кнопок/инпутов.
- Компоненты: хедер/навигация, карточки мест/маршрутов/активностей, модалки/тосты, чат-виджет.

Сохранять версионированные экспорты (`design/exports/…`) и UX-спеки (`design/specs/…`).
